===========================================================
:mod:`theano.tensor.nnet.spatialtf` -- Spatial Transformer
===========================================================

.. warning::

    Currently:

     - Only 2D inputs are supported
     - Only bilinear interpolation is implemented as the sampling method.
     - Gradients that depend on grid are not well computed on CPU.

.. note::

   This interface is the prefered interface. It will be moved
   automatically to the GPU.
